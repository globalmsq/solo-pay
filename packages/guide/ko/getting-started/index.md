# 개요

SoloPay는 블록체인 결제 게이트웨이입니다. 가맹점이 ERC-20 토큰 결제를 쉽게 받을 수 있도록 API와 SDK를 제공합니다.

## 주요 기능

### 결제 API

- 결제 생성 및 고유 ID 발급
- 실시간 결제 상태 조회
- 결제 내역 조회

### Gasless 결제

- 사용자가 가스비 없이 서명만으로 결제
- ERC-2771 메타트랜잭션 표준 기반
- Relayer가 트랜잭션 대신 전송

### Webhook

- 결제 상태 변경 시 실시간 알림
- 서명 검증으로 보안 강화

## 결제 플로우

1. **결제 생성**: 가맹점 서버에서 SoloPay API로 결제 생성 요청
2. **사용자 결제**: 사용자가 지갑에서 토큰 전송 또는 서명
3. **상태 확인**: 블록체인에서 트랜잭션 확정
4. **알림 수신**: Webhook으로 결제 완료 알림
